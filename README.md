# Vidhi Patel — Personal Portfolio + Blog

ITEC 3020 web development project (Summer 2026). A multi-page personal portfolio site with a
home page, about page, resume page, projects page, blog, and contact page, built with plain
HTML, CSS, and vanilla JavaScript.

## Repo structure
- `index.html`, `about.html`, `resume.html`, `projects.html`, `blog.html`, `contact.html` — pages
- `css/` — stylesheets
- `js/` — theme toggle, component loader, blog renderer, form validation, filter logic
- `components/` — shared header/footer HTML fragments, loaded dynamically
- `data/posts.json` — blog post content
- `images/` — photos, icons, assets

## Built in phases
- Phase 1: HTML/CSS foundation (all required pages, responsive layout, custom styling)
- Phase 2: JavaScript enhancements (theme switcher, dynamic header/footer, dynamic blog,
  form validation, live filtering, dev-log blog posts)
